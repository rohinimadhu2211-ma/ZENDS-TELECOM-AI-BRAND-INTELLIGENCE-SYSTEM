# ZENDS-TELECOM-AI-BRAND-INTELLIGENCE-SYSTEM
ZENDS Telecom AI Brand Intelligence System is an enterprise-level Artificial Intelligence platform developed for analyzing telecom customer feedback automatically using Natural Language Processing, Deep Learning, Retrieval-Augmented Generation, and interactive business intelligence dashboards.  
The main objectives of this project are:

• Generate synthetic telecom customer feedback datasets

• Build NLP-based sentiment analysis models

• Detect telecom service categories automatically

• Develop a Retrieval-Augmented Generation system

• Create enterprise-level Streamlit dashboards

• Generate telecom business intelligence insights

• Visualize telecom customer analytics using advanced charts
A virtual telecom company named ZENDS Communications was created for this project.

ZENDS Communications provides:
• Mobile Network Services
• Broadband Internet Services
• Billing and Payment Systems
• Customer Support Services
• Mobile Application Services
• SIM and eSIM Activation Services

The project simulates real-world telecom business operations and customer support workflows.

DATASET INFORMATION

The telecom dataset was synthetically generated because real telecom datasets are not publicly available.

Dataset Size:
• More than 10,000 telecom customer feedback records

Dataset Columns:
• feedback_text
• service_category
• sentiment

Supported Sentiments:
• Positive
• Neutral
• Negative

Supported Service Categories:
• Mobile Network
• Broadband Service
• Billing & Payments
• Customer Support
• Mobile App Issues
• Service Activation

DATA PREPROCESSING STEPS

• Removing duplicate records
• Converting text into lowercase
• Removing punctuation
• Tokenization
• Stop-word removal
• Cleaning noisy text
SENTIMENT ANALYSIS

DistilBERT transformer model from HuggingFace was used for telecom sentiment analysis.

The model classifies telecom customer feedback into:
• Positive
• Neutral
• Negative

The model was fine-tuned using the telecom synthetic dataset.
TOPIC CLASSIFICATION

The project automatically detects telecom service categories from customer feedback.

Supported categories include:
• Mobile Network
• Broadband Service
• Billing & Payments
• Customer Support
• Mobile App Issues
• Service Activation
RETRIEVAL-AUGMENTED GENERATION (RAG)

A RAG system was developed using:
• Sentence Transformers
• ChromaDB Vector Database

Telecom policies and FAQ documents were converted into vector embeddings and stored in ChromaDB.

The system retrieves relevant telecom information and generates AI-powered customer assistance responses.
